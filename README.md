## Getting started

##### You have a working [Go environment].

```
mkdir -p $GOPATH/src/deb-ict
cd $GOPATH/src/deb-ict
git clone https://github.com/deb-ict/cloudbm-community
cd cloudbm-community
go run ./cmd/webhost
```
